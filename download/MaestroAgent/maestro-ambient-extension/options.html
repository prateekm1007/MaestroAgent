Maestro Ambient Whisper — Settings

Configure the Maestro API URL. The extension will call the Whisper API at this address when it detects you're in Gmail, Calendar, GitHub, Zoom, or Slack.

Privacy: This extension does NOT inspect email bodies, document content, or keystrokes. It uses only the page URL and title to infer context, then calls the Maestro Whisper API. No data is stored or transmitted elsewhere.
